#### 以下为使用预设的注意事项：
- 改灰魂基础设定/改掉灰魂改成自己的角色的话改预设头部那里的设定
- 负面阴暗和消极黑暗可以同时开
- 兼容性不是很好，如果角色卡自带cot请关闭预设cot

### 预设教程
魔改预设的话新建一个放着你的要求的扔到`<order></order>`里面就可以

本预设推荐使用酒馆自动解析思思维链：
1. 找到酒馆里长得像A标志的按钮
2. 打开自动解析
3. 新建一个解析规则
4. 把标签改为 `<thinking>` 和 `</thinking>`
